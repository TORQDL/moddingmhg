A Note About Paths and Folders

MHG Root Folder

This document will refer to your Making History Gold root folder as being the folder in which the Modern Europe folder exists. From here on out, it will be referenced as .\ when providing paths to files.

The MHG root folder will likely exist at one of the following locations, depending on where you obtained your copy of the game from.

  • If you purchased the game from Steam:
    • C:\Program Files (x86)\Steam\steamapps\common\Making History Gold
  • If you purchased the game directly from Muzzy Lane or Factus Games:
    • C:\Program Files (x86)\Muzzy Lane Software\Making History Gold
  • When working in Wine for macOS, your data may end up in multiple directories.
    • Your default game data and scenarios will end up here:
      ~/.wine/drive_c/Program Files (x86)/Muzzy Lane Software/Making History Gold
    • Your output files, such as the options.xml and game saves will end up here:
      ~/.wine/drive_c/users/Public/Application Data/Muzzy Lane Software/Making History Gold

Scenario Root Folder

This document will refer to your scenario's root folder as being the folder which holds the code for your scenario, which will be located in .\Modern Europe\system\scenarios and titled as whatever you named it. From here on out, it will be referenced as $\ when providing paths to files.

Saved Games Folder

This document will refer to your saved games folder as being the folder in which the game stores your saved games, which will be located in yur username folder inside of .\Output Files\Saves. From here on out, it will be referenced as ₲\ when providing paths to files. But we probably won't be refering to saved game files every again after this.

Custom Display Settings

If you would like to change your display resolution to a setting not provided by the game's default settings, you can edit the options.xml file located at .\Options\options.xml

<?xml version="1.0" encoding="UTF-8"?>
<objects name="Object Load" version="0.1" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <object type="SystemOptions" name="">
        <property name="nScreenWidth" value="1920"/>
        <property name="nScreenHeight" value="1080"/>
        <property name="bMuteSoundEffects" value="true"/>
        <property name="bMuteMusic" value="true"/>
    </object>
</objects>

Mute

You can also toggle your audio settings in the options.xml file.

<?xml version="1.0" encoding="UTF-8"?>
<objects name="Object Load" version="0.1" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

        <property name="bMuteSoundEffects" value="true"/>
        <property name="bMuteMusic" value="true"/>
    </object>
</objects>

Building a Great Nation

Changing a Nation's Color

To change a nation's color, open the gamestate.xml file located in the root of your scenario folder.

The property named colCountryColor has the current color used for the particular country. Enter a new HEX value for the county's new color. Be sure to leave the FF at the end of the color code.

<?xml version="1.0" encoding="UTF-8"?>
<objects name="Object Load" version="0.1" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <object type="GameState" name="">
        <object type="Player" name="objPlayers">
            <property name="strName" value="Afghanistan"/>
            <property name="bNonPlayable" value="true"/>
            <property name="colCountryColor" value="#30302EFF"/>
        </object>
        
        ...
        
    </object>
</objects>

Running on macOS

A Window to a New World

So you have a Mac, but you miss playing the classic Gold Edition? Now you can with a little help from Wine. Just don't let the French know.

Wine Is Not an Emulator

Wine is a free and open-source compatibility layer for running Windows applications on Unix-like operating systems, including macOS. It has only two prerequisites for running on macOS:

  • XQuartz >= 2.7.7
  • Gatekeeper must not be set to block unsigned packages.

XQuartz is a windowing system that allows cross-platform applications to run on macOS. It is sometimes refered to as X11, X, or X-Windows.

Gatekeeper is a security feature of macOS that enforces code signing and verifies downloaded applications before allowing them to run. It reduces the likelihood of inadvertently executing malware. If you have changed your Gatekeeper settings already, then you likely know what you're doing. If you have not, the default settings for Gatekeeper can be overcome by a simple right-click on the mouse, which will be outlines in the instructions below.

Because this is a more advanced topic, and you should only be taking this on if you feel comfortable using your computer in an advanced way, these instructions will not be extremely detailed. It is assumed that if you are doing this, you can read between the lines and understand how to install and run software on your Mac.

Installing Wine 2.x

In order to play Making History Gold on macOS, we need to be able to run Windows software on macOS. To do this, we will install Wine 2.x and XQuartz.

XQuartz

One of the prerequisites of running Wine on macOS is XQuartz; therefore, we will install it first.

Download and install the latest version of XQuartz from here: XQuartz.org

Before you install XQuartz, be sure to save and close all open applications. A logout or restart is required after XQuartz installation.

Wine

Download and install the latest stable release of Wine from here: WineHQ

Post Installation

If you already tried to launch Making History Gold, you likely found it unusable. This is because we are missing a couple of components:

  • DirectX
  • Fonts
  • Multiplayer

DirectX

DirectX is required for Making History Gold to run. It's the 3D graphics and audio framework. Wine 2.x has basic DirectX support built in, but you need the full DirectX 9.0c library for Making History Gold to run properly.

  • You can download it directly from Microsoft, or you can download it from me. Either way, you should be getting the same file.
  • Download from Microsoft: DirectX
  • Download from here: DirectX
  • MD5 (directx_Jun2010_redist.exe) = 7c1fc2021cf57fed3c25c9b03cd0c31a
  • Because you have Wine installed, directx_Jun2010_redist.exe should have a Wine icon. Double-click directx_Jun2010_redist.exe to open it in Wine and follow the installation wizard.

Fonts

The fonts used inside of Making History Gold may be default fonts, but the fonts built into macOS don't seem to work with it. This is likely due to the version or format of the fonts under macOS. These fonts are Arial and Courier. I have exported them from Windows and zipped them up for your convenience.

  • Download the fonts: ZIP 
    MD5 (winefonts.zip) = fe634bddc6efd8b7c827a0c5c555814d
  • Once you have the fonts downloaded and extracted, place them in this folder:
    • ~/.wine/drive_c/windows/Fonts
  • You should have the following nine font files:
    • arial.ttf
      arialbd.ttf
      arialbi.ttf
      ariali.ttf
      ariblk.ttf
      cour.ttf
      courbd.ttf
      courbi.ttf
      couri.ttf

Multiplayer

I have not yet got the network stack running properly, so trying to play a multiplayer game under Wine will cause the game to crash or freeze.
