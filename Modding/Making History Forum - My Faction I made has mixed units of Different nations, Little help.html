Dude, I can't believe no one answered you any sooner. But I can actually help ya with that one. What you have to do is go to this file called "TableCountriesAlliansesLand" (or at least I think that was the exact name). I think you can find this file in Display Libraries/Sprite Maps. Not sure about the exact directory as I'm currently at work and can't check for you lol. But I'm sure you'll find it. Open the aforementioned file with notepad and you'll see the names of all the nations followed by a lot of codes. What I would suggest you do is pick one country who's units you'd like your country to have and copy the whole code under the name of that country. Then, change the name of the country in the copied code to the name of your country. If you don't want your country to have the exact same units as that of the country who's code you copied then you'll have to change a few things here and there in the code. Say you copy the USSR code and you wanna keep the soviet infantry but want to have German tanks instead. You scroll through the whole code and in places where it says "USSR light tank" or "USSR Heavy tank" you change the "USSR" to "German". And voila! The same method is used for changing air units only instead of "land" the file name ends with "air". Hope this helped.
