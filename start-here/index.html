Overview

In March of 2007, Muzzy Lane released a new educational game called Making History: The Calm & the Storm. Similar to games like Axis & Allie and Risk, Making History is a turn-based grand strategy game centered around a WWII theme with a basic industrial, economic, resource, research, and diplomacy management system. The game includes a scenario editor, allowing players to build their own custom scenarios for expanded gameplay. In July of 2008, a Gold Edition of the game was released, featuring new scenarios and combat concepts, such as partisan insurgencies, separate attack and defense strengths, and numerous enhancements. It has become the de facto version that most players prefer. Since then, multiple sequals to the game have been release, but this site is dedicated to Making History: The Calm & the Storm Gold Edition.

In early 2016, the lead designer of the Making History games started Factus Games, and purchased the publishing and development rights to the Making History series. In addition to the Making History series, Factus Games is also planning a new line of turn-based strategy games that explore a range of historical eras.

Throughout this site, Muzzy Lane and Factus Games may be used interchangably, although they are not the same company. The game was originally published under Muzzy Lane, so a lot of reference to Muzzy Lane is present.

Where are my files?

Depending on how you aquired your game, whether directly from Muzzy Lane or Factus Games, or if you bought it on Steam, your game files may be located in different locations. Below is a list of places to look for your files, depending on how you installed it.

Direct Purchase

You got your game direct from Muzzy Lane or Factus Games? If you're using anything newer than Windows XP, your files are going to be spread out in multiple places. This is due to Windows security settings that now prevent normal usage of the Program Files directories.

For the sake of this site, I'm going to make a few assumptions. First off, I'm assuming you're not using Windows XP. If you are, you probably shouldn't be using a computer anyway. Second, I'm assuming you're using a 64-bit operating system. If you're using a 32-bit operating system, and using this site, I'm going to assume you know the difference between Program Files and Program Files (x86) already, and you can follow along without additional help.

Installation Location

Your game files will install to this directory by default:
C:\Program Files (x86)\Muzzy Lane Software\Making History Gold

Output Files

Your output files, such as your options settings and saved games, will save to this directory:
C:\ProgramData\Muzzy Lane Software\Making History Gold

Scenarios

The scenarios folder will default to the correct location under the Program Files. When you install a custom scenario, this is where you will also put it.
C:\Program Files (x86)\Muzzy Lane Software\Making History Gold\Modern Europe\system\scenarios

However, when you use the editor to create or edit scenarios, it will save the scenario files in a different directory:
C:\Users\<username>\AppData\Local\VirtualStore\Program Files (x86)\Muzzy Lane Software\Making History Gold\Modern Europe\system\scenarios

Steam Purchase

If you purchased the game on Steam, everything will install to the same location, without Windows security moving things around. All files that you need can be found under your Steam library directory, which defaults to this directory:
C:\Program Files (x86)\Steam\steamapps\common\Making History Gold

Wine on macOS

If you are using the direct installer from Factus/Muzzy Lane, and are installing in Wine on macOS, your files will end up in these locations.

Installation Location

Your game files and scenarios will all end up in the default location:
~/.wine/drive_c/Program Files (x86)/Muzzy Lane Software/Making History Gold

Output Files

Your output files, though, will end up in another location:
~/.wine/drive_c/users/Public/Application Data/Muzzy Lane Software/Making History Gold
